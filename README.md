# helper-web

---

## 技术栈

工程技木栈：

- Node.js 20+
- TypeScript 5.4+
- Nuxt.js 3.0+
- NuxtUi 2.0+
- tailwindcss

参考：

- [Nuxt: The Intuitive Vue Framework · Nuxt](https://nuxt.com/)
- [Nuxt UI: A UI Library for Modern Web Apps](https://ui.nuxt.com/)

---

## 开始

```shell
# 安装依赖
npm install

# 开发模式
npm run dev
```
